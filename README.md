# Itero-TC-Tracker

A high-precision Health & Safety (H&S) training compliance management system with a premium, state-of-the-art interface.

## 🚀 Vision
Built to eliminate data friction, the Itero-TC-Tracker provides a "Front Door" for all training operations, ensuring audit readiness and proactive renewal management through:
- **Multi-Selection Log Entry**: Batch process records for teams.
- **Credential Vault**: Secure storage for external training portal logins.
- **Horizon Monitoring**: Color-coded tracking of training expirations.
- **ROI Analytics**: Real-time spend and compliance reporting.

## 🎨 Design System
Inspired by high-end financial trackers, utilizing:
- Glassmorphism & Backdrop blurs.
- Midnight base with Electric Purple / Cyan accents.
- Modern Inter typography.

## 🛠️ Tech Stack
- **React 18** + **Vite** (Build System)
- **Vanilla CSS** (Custom Design Tokens)
- **State Management**: React Hooks

## 🏁 Getting Started
1. `cd Itero-TC-Tracker`
2. `npm install`
3. `npm run dev`
